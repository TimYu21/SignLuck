# Changelog

Все изменения проекта "SignLuck" будут документироваться в этом файле.

Формат базируется на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2024-02-23

### Изменено

#### Backend
- **Refactoring**: Реструктуризация моделей данных (Schemas).
  - Монолитный модуль `plate.py` разделен на тематические файлы:
    - `country.py` — модели стран и репозитория.
    - `trip.py` — модели маршрутов и сегментов путешествия.
    - `search.py` — модели API запросов и ответов поиска.
  - Улучшена читаемость кода и разделение ответственности.


## [1.0.0] - 2024-02-22

Первый стабильный релиз системы анализа вероятности автомобильных номеров.

### Добавлено

#### Backend
- **Core Logic**: Реализован `PlateCalculator` для точного расчета вероятности выпадения номера с учетом шаблонов стран и спецсимволов.
- **Services**:
  - `PlateService`: Оркестрация поиска, фильтрация результатов, сортировка по вероятности.
  - `CountryRepository`: Ленивая загрузка базы данных стран из CSV, генерация эмодзи флагов.
- **Features**:
  - Алгоритм построения "Маршрута удачи" (Nearest Neighbor) с учетом геопозиции пользователя.
  - Генерация deep-link ссылок на Google Flights для покупки билетов.
  - Поддержка Leet-speak замен (например, 'O' -> '0') для генерации подсказок.

#### Frontend
- **Framework**: Vue 3, TypeScript, Vite.
- **UI/UX**:
  - Адаптивный интерфейс в стилистике Cyberpunk / Glassmorphism.
  - Анимированные переходы и состояния загрузки.
- **Components**:
  - `ProbabilityMap`: Интерактивная карта мира (Leaflet + GeoJSON) с тепловой картой вероятностей.
  - `RouteMap`: Визуализация маршрута путешествия с анимированными линиями.
  - `SearchInput`: Умный поиск с валидацией и автодополнением.
  - `TripRoute`: Отображение сегментов перелета с кнопками покупки.
- **Localization**: Полная локализация интерфейса (Русский / Английский).
